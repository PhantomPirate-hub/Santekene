🚀 DÉMARRAGE ULTRA-RAPIDE (5 MINUTES)
=======================================

1. INSTALLER LES PRÉREQUIS
   - Node.js 20+ : https://nodejs.org/
   - MySQL 8.0+ : https://dev.mysql.com/downloads/
   - Python 3.11+ : https://www.python.org/downloads/

2. INSTALLER LE PROJET
   ```bash
   # Backend API
   cd backend-api
   npm install
   cp .env.example .env
   # Éditer .env avec vos configs

   # Créer la BDD
   mysql -u root -p
   CREATE DATABASE santekene;
   EXIT;

   # Migrations
   npx prisma migrate dev
   npx prisma db seed

   # Backend AI
   cd ../backend-ai
   python -m venv venv
   .\venv\Scripts\Activate.ps1  # Windows
   pip install -r requirements.txt

   # Frontend
   cd ../frontend
   npm install
   cp .env.example .env.local
   ```

3. LANCER LES 3 SERVICES (3 TERMINAUX)
   
   Terminal 1 - Backend API :
   ```bash
   cd backend-api
   npm run dev
   ```
   ✅ http://localhost:5000

   Terminal 2 - Backend AI :
   ```bash
   cd backend-ai
   .\venv\Scripts\Activate.ps1
   python main.py
   ```
   ✅ http://localhost:8000

   Terminal 3 - Frontend :
   ```bash
   cd frontend
   npm run dev
   ```
   ✅ http://localhost:3000

4. SE CONNECTER
   URL : http://localhost:3000
   Email : superadmin@santekene.com
   Mot de passe : SuperAdmin2024!
